Lava-Typers
PROJECT DETAILS

Lava-Typers

  • Project 2

  • December 2016

The goal of this game is to be able to type out the word correctly before it reaches the lava. The player will have 5 lives to start with and will lose a life every time the word falls into the lava. Once all of the player's lives are used up, the game will be over. Data will be recorded only if the user is logged in where they can click on their profile to check their highest score, average words per minute and average accuracy. This app was built using Ruby on Rails as back end, and CSS, Bootstrap and Javascript for the front end
